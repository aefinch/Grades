# Student Grades

### Description
This is a practice exercise to sort arrays and compare values based on ranges.

### Technologies Used
- HTML
- CSS
- Javascript

### How to Run
1. git clone [here](https://github.com/aefinch/Grades)
1. cd to the file location
1. Run http-server
1. Navigate in browser to http://localhost:8080
